# Order Page Design System Override

## Multi-Step Form
Step 1: EVENT BASICS — Event type dropdown, date picker, servings slider
Step 2: CAKE PREFERENCES — Flavor checkboxes with icons, icing type cards with images, dietary needs toggles (gluten-free, vegan, dairy-free)
Step 3: DESIGN VISION — Image upload zone, description textarea, color palette selector
Step 4: CONTACT — Name, email, phone, delivery/pickup radio buttons, special instructions

## UX Requirements
- Progress bar at top showing current step
- Step validation before advancing
- Form data persisted to localStorage
- On submit: redirect to Stripe Payment Link for deposit
- "Save for Later" option
